Inserindo videos hospedados localmente

Este vídeo está hospedado no meu próprio servidor.

Para inserir um video hospedado no próprio servidor precisamos usar a tag "video" com as tags "width" para o tamanho na página, "poster" para colocar uma tumb no video, "controls" para aparecer os controles do video e "loop". Também temos que usar dentro da tag os "source src" para assim introduzirmos diferente tipos do arquivo de video para rodar em todos os navegadores possíveis.

Inserindo videos do YouTube

Para inserir videos do YouTube é necessário ir no video em compartilhar e clicar em incorporar, após isso é só colar no código html
