Trabalhando com Listas

Listas Ordenadas

Para criarmos listas ordenas primeiro usamos a tag "ol" para declarar a lista enseguida usamos a tag "li" para declarar os itens da lista. Para mudar o estilo da lista usamos dentro da tag "ol" o parametro 'type'.

  1. Acordar
  2. Ligar a luz
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Ir para o trabalho
  7. Voltar para casa
  8. Jantar
  9. Dormir

Listas não Ordenadas

Para criarmos uma lista não ordenada usamos a tag "ul", seguido da tag "li" para declarar seus itens. Também podemos mudar o estilo da lista usando o parametro 'type' com os formatos "circle", "disc" e "square".

  • Pão
  • Leite
  • Frios
  • Tomate
  • Oregano
  • Batata
