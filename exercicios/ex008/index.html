Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>(tag não semântica)

Nesta frase temos um termo em destaque usando a tag STRONG (Uma tag que possui semântica significado)

Itálico ou enfase

Nesta frase temos um termo em itálico usando a tag (I) (não semântica)

Nesta frease temos um termo em enfase usando a tag EM (com semântica e significado)

Texto marcado

Podemos criar também um texto marcado com a tag MARK

E no outro parágrafo, temos outro texto marcado no final

Para mudar a cor da marcação é indicado usar o "style" em css para mudar a forma

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.(semântica)

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag "sup" inserimos o ao quadrado, cubo, etc.

Texto subescrito

Para inserir coisas do tipo H2O com a tag "sub" conseguimos inserir o log, e números de formúlas químicas abaixo.
